ライブイベントのスケジューリング

{{page.description}}

ライブモジュールは、繰り返しイベントが開始および終了するタイミングをスケジューリングする機能を提供します。スケジュールされた開始時刻に達すると、繰り返しイベントがアクティブになります。スケジュールできるのは、繰り返しイベントとして作成されたイベントだけです。

注:現在、冗長グループのスケジューリングはサポートされていません。

スケジュールされたイベントに関する注意事項

  • 一度に6つの保留中のアクティベーションをスケジュールできます。さらに必要な場合は、Brightcoveサポートにお問い合わせください。
  • イベントがアクティブ化されると、デフォルト再接続時間は引き続き有効です (参照ライブ API:詳細については、静的エントリ ポイント)
  • 関連するすべての制限静的エントリ ポイントまだ有効です

イベントへのスケジュールの追加

アカウントで自動開始/停止が有効になっている場合、スケジューリングこのセクションは、イベントを編集するときに使用できます ([コントロール ルーム] ページで)。

自動開始/停止を使用するには、以下の手順に従ってください。

  1. を開きますライブモジュール。
  2. 新しい定期的なライブ イベントを作成するか、待機するタブをクリックし、既存のイベントをクリックします。
  3. 上で制御室ページで、スケジューリングセクション。
  4. クリック+ スケジュールを追加そしてイベントに入る開始日と終了日.
    注:複数のスケジュールが同時に実行されるのを防ぐことはできませんが、競合する行には警告が表示されます。競合するスケジュールは、アクティベーション時間より前に対処しないと失敗する可能性があります。警告を回避するには、行の1つを削除するか、時間を調整します。
  5. 別のスケジュールを作成するには、 + スケジュールを追加.の+ スケジュールを追加保留中のアクティブ化の最大数に達すると、ボタンは無効になります。
  6. 注意:ライブUIで表示/作成できるスケジュールには、6つのソフト制限があります。ただし、さらに必要な場合は、BrightcoveSupportに連絡してこの制限を増やすことができます。

    スケジュールを削除するには、削除 ( ) アイコンをクリックします。

    注意:
    • スケジュールが衝突すると、スケジュールされたイベントが警告として黄色に変わります。
    • 何らかの理由でストリーミングに失敗すると、スケジュールされたイベントが赤に変わります。
    • 失敗したスケジュールは7日間表示されます。

今後のアクティベーションの表示

スケジュールされたすべてのイベントを表示するには、待機するタブ。次の予定始めると止まる日付は今後題名。
