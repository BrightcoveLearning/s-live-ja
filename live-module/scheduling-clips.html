In this topic, you will learn how to schedule a video clip using the Live module.

Scheduling a clip in the Control Room

  1. Open the Live module.
  2. Create a new Live event with a Static Entry Point or visit a previously created one.
  3. In the Control Room, scroll down to the Scheduled clips section.
  4. Click Add Scheduled Clip to schedule a new clip.
    Note: If the Scheduled Clips panel is not available in the Control Room, please contact Brightcove support to enable it for you.
  5. A modal window will appear. Select the Start Date and Time and the End Date and Time
      Note:
    • The Start Date and Time and End Date and Time of your scheduled clip should match a current or future stream event. End Date must be in the future. Otherwise, the clip creation will fail.
    • The minimum duration for a scheduled clip is 5 minutes and the maximum 24 hours.
  6. The Duration of the clip is automatically calculated.
  7. Give it a Name.
  8. Write a Short Description. (Optional)
  9. Assign Tags if necessary. (Optional)
  10. Add a Reference ID. The ID must be a unique value for the entire Studio account. (Optional)
  11. Click Schedule Clip.
    12. The scheduled clip will be created when the End Date is reached. Meanwhile, an empty video asset will be created as a placeholder in the media module.
  12. To update a scheduled clip, click on the clip.
  13. Make the desired changes.
  14. Click Update Clip.
  15. To delete a scheduled clip, click the trash can icon.

Scheduling a clip in the clipping page

  1. In the Control Room, click Create Clip from the left menu.
  2. Click Add Scheduled Clip to schedule a new clip.
    Note: If the Scheduled Clips panel is not available in the Control Room, please contact Brightcove support to enable it for you.
  3. A modal window will appear. Select the Start Date and Time and the End Date and Time
      Note:
    • The Start Date and Time and End Date and Time of your scheduled clip should match a current or future stream event. End Date must be in the future. Otherwise, the clip creation will fail.
    • The minimum duration for a scheduled clip is 5 minutes and the maximum 24 hours.
  4. The Duration of the clip is automatically calculated.
  5. Give it a Name.
  6. Write a Short Description. (Optional)
  7. Assign Tags if necessary. (Optional)
  8. Add a Reference ID. The ID must be a unique value for the entire Studio account. (Optional)
  9. Click Schedule Clip.
    10. The scheduled clip will be created when the End Date is reached. Meanwhile, an empty video asset will be created as a placeholder in the media module.
  10. To update a scheduled clip, click on the clip.
  11. Make the desired changes.
  12. Click Update Clip.
  13. To delete a scheduled clip, click the trash can icon.
  • You only will be able to update the scheduled clip Start Date and End Date only before the end date is reached.
  • Red entries may appear in the Scheduled Clip panel, those are Failed schedules. Scheduled clips can fail for a variety of reasons, for example, cue-points that don't fall within the VOD start and end times.
