Supported Encoders for Live Events

This topic is intended to provide some guidelines for hardware encoder settings when conducting a live stream event. For a list of all the options available from Brightcove for delivering live streaming video, see {% if site.product_short == "Video Cloud" or site.product_short == "Beacon" or site.product_short == "CorpTV" %} Delivering Live Streams. {% endif %} {% if site.product_short == "Live" %} Delivering Live Streams. {% endif %}

The following is a list of encoders that are generally known to work with Brightcove Live. Other encoders may also work; this list contains the encoders that we have specifically tested.

Also note that supported does not mean that every feature is supported. Specific capabilities (such as sending cuepoints) vary among different encoders.
  • XSplit
  • OBS
  • Wirecast
  • Cambria Live
  • Liveshell
  • AWS Elemental Live
  • Haivision
  • LiveU
  • Matrox
  • NewTek
  • Teradek
