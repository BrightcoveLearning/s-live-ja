Brightcoveのライブストリーミングビデオ機能により、どのデバイスやブラウザにも高品質のライブストリーミングブロードキャストを配信できます。ライブストリームの配信には、カスタム開発または複雑な統合は必要ありません、独自のニーズがある、カスタム統合が必要な場合には、APIを利用できます。

注:ライブストリーミングのサポートは、Enterpriseパブリッシャーでのみ利用できます。

BrightcoveはDVRによるライブストリーミングもサポートしています。Brightcove Player がDVRUX プラグイン、ライブ ストリームは、視聴者のデジタル ビデオ レコーダーで再生されているかのように機能します。

Brightcoveは、ライブストリーム配信用のオプションを複数提供しています。このトピックでは次の項目について説明します。

  • ハードウェアとクラウドベースのエンコーディング
  • ライブモジュールを使用したライブストリーミング
  • ハードウェアベースのエンコーダーを使用したライブストリーミング

ハードウェアとクラウドベースのエンコーディング

Brightcoveは、ハードウェアベースとクラウドベースの両方のエンコーディングをサポートしています。イベントにハードウェアベースとクラウドベースのどちらのエンコーディングが適しているかは、さまざまな要因で決まります。

ハードウェアエンコーディング使用

  • 最大容量が明確な24x7のリニアフィード
  • ロケーションが1つのアーキテクチャー
  • ブロードキャストフォーマットのサポート

クラウドエンコーディング使用

  • イベント用
  • 容量の要件が分からないとき
  • 分散型の組織向け
  • イベントのタイミングが不明で、配信開始までのスピードが重要なとき
  • 帯域幅ボトルネックがある場合

ライブ API

ライブ API は、ライブストリーミングイベントを作成および管理できる REST ベースの API です。オプション機能は次のとおりです。

  • サーバー側広告挿入 (SSAI)
  • AES-128暗号化
  • ライブストリームから取得したクリップからビデオオンデマンドアセットを作成する
  • DVR機能
  • 複数の CDN

詳細については、ライブ API ドキュメント.

ハードウェアベースのエンコーダーを使用したライブストリーミング

このオプションでは、ハードウェアベースのエンコーダーを使用して、ライブストリームをエンコードし、お客様のCDNへ配信します。Video Cloud側では、ストリームをホストするためのリモートアセットを作成します。その後、ライブストリーム用のプレーヤーを公開できます。
